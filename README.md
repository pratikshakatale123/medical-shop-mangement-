
# Medical Shop Management System

A Java Swing + MySQL based desktop application for managing medical shop inventory.

## Features
- Secure Login System
- Medicine Inventory Management
- Add / Update / Delete Medicines
- Search Functionality
- JDBC Database Connectivity

## Technologies Used
- Java (Core Java, Swing)
- JDBC
- MySQL
- SQL

## Database Setup
1. Open MySQL Workbench
2. Create a new database
3. Import the file:
   database/medical_management.sql

## Note
- Update database username and password in ConnectionDB.java before running.
